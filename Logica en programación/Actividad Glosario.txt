GLOSARIO DE TÉRMINOS 

SWITCH: Es un aparato que permite la conexión entre dispositivos, para la comunicación entre estas mismas. 

HUB: 	El HUB es la forma inteligente hablando de la red, que es comparado con el Switch, pero la diferencia de que es mediante una red,  el cual podemos conectar varios aparatos entre sí para la comunicación,

ROUTER : Es el aparato que nos provee acceso a las red local, a internet mediante una forma inalámbrica mediante cualquier aparato tecnológico ( tablet, impresoras, discos de almacenamientos, etc.. )

AP: 

PROXY: Es la parte que sirve como intermedio de una forma entre un objeto o dispositivo y una servicio o red.

FIREWALLl: Es un sistema que resguarda o protege a nuestra red privada, de usuarios o intrusos que tengan malas intenciones, como un virus. 

RACK: Es una armario para guardas dispositivos tecnológicos, que sean importante para el uso personal y privado. 

UPS: Es un dispositivo de almacenador de energía, para uso durante un apagón eléctrico en un tiempo estimado a su capacidad.

SAN: Es un área de almacenamiento que utilizan algunas empresas para obtener un alto rendimiento, y también se puede decir que consta de un sistema de seguridad y recuperación de datos en un caso desastroso. 

BACKUP: En simple palabras es una copia de seguridad de los datos originales, en caso de una perdida que pueda suceder. 

RESTORE: Es un sistema para restaurar ficheros que se usa en Microsoft  ( MS-DOS  ) 

CENTRO DE CABLEADO: Es un lugar para la conexión de dispositivos en una red local. 

CENTRO DE DATOS: Es el lugar o espacio donde se procesa la información mediante unos recursos necesarios para este proceso,en una organización.

PLANTA TELEFÓNICA: Es una central que lo que permite es la comunicación, interconexión de diferentes grupos de teléfonos. 

ETHERNET: Es una red local usada en las computadoras.

TARJETA CONTROLADORA: Es una pieza que se utiliza como interfaz entre la placa base y los otros componentes del computador.  

UTP: Es un cable que consiste en grupos de hilos que están entrelazados helicoidal, que conforman una antena simple. 

STP: Es un sistema que se encarga de administrar y reconocer bucles de lugares de redes. 

CABLE CRUZADO: Es un cable que conecta dos dispositivos similares mediante un cable modificado llamado “crosslink”. 

CABLE COAXIAL: Es un cable que consiste en transportar corrientes eléctricas de alta frecuencia, que poseen dos conductores, uno central llamado (núcleo) y otro externo llamado. (malla)

FIBRA OPTICA: las fibras se utilizan para transmitir luz y la transmisión en largas distancias y no les hace efecto la energía electromagnética. 

CABLEADO ESTRUCTURADO: Es un sistema de cable, formado por un par trenzado protegidos por STP o UTP.

HOST: Se refiere a una serie de dispositivos (tablests, moviles etc.), que usan un servidores que poseen anfitriones los cuales administran una red local que están conectados a la red. 

HOSTING: Es un servicio de alojamiento en guardar información (videos, imágenes, etc.) en un sitio web. 

DNS: Es un servidor que consiste en dar una respuesta a varias peticiones de usuarios o clientes en el ámbito del internet.

DOMINIO: El propósito del dominio en internet es dar un nombre a las direcciones web para una mayor facilidad en la búsqueda de un usuario en el navegador web, ejemplo 
http://172.217.10.110/, en vez de http://google.com)

MAC: Es una dirección que se utiliza en 48 bits para las computadoras.

PROTOCOLO ARP: Es un protocolo de comunicaciones en la capa de datos, para resolver la dirección de IP de una dirección hardware. 

PROTOCOLO TCP Y UDP: Es un protocolo que transporta datagramas,a partir de la conexión de dos maquinas.

IP: Se emplea para el envió de información mediante una red, y hace referencia al equipo de origen. 

IPv4: Es una versión del internet implementada en 1983. 

IPv6: Es una versión mas sofisticada para arreglar problemas existentes en la anterior versión. 

TCP: Es un protocolo el cual consiste en la comunicación de computadoras mediante una red local.

PING ( ICMP): Es un programa que sirve para comprobar el estado de la comunicación y para medir el tiempo remoto que tarda en comunicarse dos puntos.

IPCONFIG / IFCONFIG: Es la aplicación que configura los valores de IP/ TCP.
Se utiliza para administrar el sistema para consultar, controlar en una red. 

MODELO TCP/ IP: Es un protocolo que fue implementado en la red ARPANET, en el cual consiste permitir que un equipo pueda comunicarse en una red. 

ANCHO DE BANDA: Se refiere a la capacidad de medida de datos y recursos de comunicación.

TOPOLOGIA  DE REDES: Es un mapa físico o lógico, el cual representa el intercambio de datos.

PROTOCOLO ENRUTADO: Es el establecimiento de direcciones en una red.

PROTOCOLO ENRUTAMIENTO: Es la construcción de tablas de las direcciones en la red para identificarlas.

INTERNET: Es un servicio de protocolos que ofrece una diversidad de recursos en un sitio web o cualquier plataforma.  

LAN/MAN/WAN: 
    • Permite la interacción de múltiples equipos para compartir datos.
    • Consiste en lo mismo pero hay una pequela diferencia es que permite compartir mayor recursos que WAN 
    • Se utiliza para abarcar mucho terreno por sus nodos que están separados.

WiFi: Es usado para la interconexión inalámbrica de dispositivos electrónicos.

ALGORITMO: Es un conjunto de instrucciones para solucionar un problema. 

VARIABLE: Es el valor que puede tomar una cantidad descrita.

CONSTANTE: Es el valor que no puede ser alterado o cambiado.

PRUEBA DE ESCRITORIO: Consiste en dar valores a un flujo del programa para comprobar si al final el resultado es el acertado.

CONDICIONAL IF: Se usa para tomar decisiones en operación lógica. 

ESTRUCTURA FOR: Es una sentencia para la iniciar el bucle. 

ESTRUCTURA DO WHILE:  Es un control repetitivo, se caracteriza en ejecutar las instrucciones sin evaluar la expresión. 

ESTRUCTURA CASO: Con esto podemos evaluar una variable y realizar acciones.

PSEINT: Es un programa para la instrucción en el mundo de la programación.
